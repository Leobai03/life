# life-sop

一个帮助年轻人建立个人生活 SOP 的免费 Codex Skill。

它不是成功学模板，也不是效率打卡表。它更像一个轻量的生活系统设计器：先帮你看清当前阶段，再把睡眠、饮食、运动、工作、内容、情绪、关系和复盘整理成一套能跑起来的日常流程。

## 适合谁

- 作息混乱，生活节奏失控的人
- 想努力，但总是启动困难、空转、自责的人
- 被赚钱、流量、结果焦虑推着走的人
- 想在 30 岁前建立一套属于自己生活系统的人
- 想先稳定身体、情绪和节奏，再长期发力的人

## 它会产出什么

使用这个 skill 后，你通常会得到：

- 一份 `个人生活 SOP v1`
- 一条当前阶段主题
- 1-3 个需要守住的优先级
- 每日最低通过线
- 起床/开始工作流程
- 主工作块安排
- 内容或学习留痕机制
- 身体底线
- 晚间收尾流程
- 情绪失控时的重启流程
- 7 天、14 天或 30 天执行清单
- 复盘模板

## 安装方式

如果你使用的是 Codex，并且本地有 skills 目录，可以把整个 `life-sop` 文件夹放到：

```bash
~/.codex/skills/life-sop
```

如果你设置了 `CODEX_HOME`，则放到：

```bash
$CODEX_HOME/skills/life-sop
```

注意：一定要上传和安装整个 `life-sop` 文件夹，不要只复制 `SKILL.md`。`references/` 里的问题库、SOP 模板和复盘模板是这个 skill 的核心组成部分。

## 使用方式

### 生活 SOP

安装后，你可以这样和 Codex 说：

```text
Use $life-sop 帮我建立一套适合当前阶段的个人生活 SOP。
```

也可以直接说：

```text
我现在作息很乱，生活没有节奏，帮我用 life-sop 做一份 14 天执行清单。
```

或者：

```text
我想建立自己的生活操作系统，请你先问我问题，然后帮我生成个人生活 SOP v1。
```

## JoeSkill 加密信号扫描

仓库内新增了一个可直接运行的 `joeskill_scan.py`，用于把 Joe 的几条加密货币筛选规则先跑起来：

- J2：昨日日成交额低于 2000 万 USDT，当前 24h 成交额大于 1 亿 USDT，寻找突然放量。
- J3：10 日内有过暴涨，现在 24h 成交额低于 1000 万 USDT，寻找暴涨后缩量潜伏。
- J4A：日线匀速 3-4 连涨，24h 成交额大于 5000 万 USDT，寻找趋势确认。

运行：

```bash
python3 joeskill_scan.py
```

输出会包含：数据源状态、Joe 主榜、今日只值得盯的 3 个、今日不要追的 3 个，以及下一次扫描建议。

说明：这是信号筛选工具，不是投资建议；所有标的都需要人工看图确认右侧量价。

## 快速输入格式

```text
1. 这阶段我最想守住的 1-3 件事：
2. 这阶段我最想得到的 1-3 个结果：
3. 我现在最混乱的 1-3 个点：
4. 我最容易失控的时间段：
5. 我现在睡眠、饮食、运动的大概情况：
6. 我一天里最有效率的时间段：
7. 我最不想继续重复的生活状态：
```

## 输出示例

```text
你的阶段主题：
稳住现金流，恢复作息节律，重新建立行动力。

每日最低通过线：
- 吃 2 顿饭
- 做 1 个直接影响现金流的动作
- 留下 1 个内容或学习痕迹
- 出门或运动 20 分钟
- 按目标时间开始收尾

复盘问题：
- 今天真正推进了什么？
- 今天最卡我的，不是“懒”，而是什么？
- 明天只需要先做哪一个动作？
```

## 设计原则

- 不追求完美生活，只追求可持续生活。
- 不把别人的目标当成自己的目标。
- 不用牺牲身体和内心平静换短期结果。
- 不把“懒”“废物”“不自律”当作最终诊断。
- 任何 SOP 都只服务于当前阶段，可以持续迭代。
